# Brand Kit - Navantia COEX

## ¿Qué es?
Este repositorio empaqueta la identidad de marca de **Navantia COEX** (Centros de Excelencia) y **COEX Digital Twin** de forma centralizada. 

## ¿Para qué sirve?
Su objetivo es garantizar la coherencia visual e identidad corporativa en todos los proyectos de aplicaciones, previniendo el uso de logotipos o estilos obsoletos. Además, está estructurado para ser fácilmente legible tanto por desarrolladores como por asistentes de Inteligencia Artificial (Copilot, Claude, etc.).

## ¿Cómo se consume en apps?
Los proyectos y aplicaciones pueden consumir este repositorio para obtener:
1. **Design Tokens** (`/design-tokens/`): Archivos JSON o compilados (CSS/TS) con variables de color, tipografía y espaciado.
2. **Assets** (`/assets/`): Logotipos e imágenes oficiales siempre actualizados.
3. **Reglas de Marca** (`/docs/brand/`): Guías de uso de la identidad corporativa.

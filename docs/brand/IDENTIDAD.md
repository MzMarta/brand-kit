# Guía de Identidad: Navantia COEX

Este documento define las reglas básicas y de uso de la identidad corporativa para los proyectos digitales de Navantia COEX, basándose en la presencia pública oficial.

## 1. Naming (Nomenclatura)
Es fundamental usar los nombres oficiales de los productos y departamentos para mantener la coherencia.

* **✅ Correcto:** Navantia COEX
* **✅ Correcto:** COEX Digital Twin
* **❌ Prohibido:** Navantia Coex, Centro Excelencia Navantia, DigitalTwin (sin espacio), Navantia DT.

## 2. Logos
Los logotipos oficiales están centralizados en este repositorio para evitar versiones obsoletas.

* **Ubicación:** `/assets/brand/upstream/`
* **✅ Permitido:** Usar el logo en positivo (sobre fondos claros) o en negativo/blanco (sobre fondos oscuros).
* **❌ No hacer:** * Estirar o deformar el logotipo.
  * Cambiar los colores corporativos del logo.
  * **Uso prohibido:** Queda estrictamente prohibido el uso del antiguo logotipo tipo "stamp" corporativo (descatalogado).

## 3. Colores (Propuesta)
*Nota: Al no contar con un manual público con los códigos HEX, esta es una **propuesta** basada en la presencia web corporativa de Navantia.*

* **Primario (Azul Marino):** `#00529C` - Para cabeceras, botones principales y fondos de énfasis.
* **Secundario (Azul Claro/Cian):** `#00A3E0` - Para detalles, acentos visuales e iconos.
* **Neutros:**
  * Blanco: `#FFFFFF` - Fondo principal.
  * Gris oscuro: `#333333` - Textos principales.
  * Gris claro: `#F4F4F4` - Fondos secundarios.

## 4. Tipografías (Propuesta)
Para garantizar legibilidad y un aspecto tecnológico, sugerimos el siguiente emparejamiento de fuentes web seguras:

* **Titulares (Headings):** *Montserrat* o *Roboto* (Transmiten solidez y modernidad).
* **Cuerpo de texto (Body):** *Open Sans* o *Inter* (Alta legibilidad en pantallas digitales).

## 5. Tono y voz
Nuestra comunicación debe reflejar el liderazgo tecnológico y naval.

**Adjetivos clave:** Innovador, Profesional, Seguro, Tecnológico, Claro.

* **✅ Hazlo (Ejemplo):** "Descubre cómo COEX Digital Twin optimiza los procesos navales."
* **❌ No lo hagas (Ejemplo):** "Mira el súper programa que hemos hecho para los barcos." (Demasiado informal).

## 6. Uso en UI (Interfaz de Usuario)
Reglas de aplicación en componentes de software y páginas web:

* **Botón Primario:** Fondo Azul Marino (`#00529C`), texto en Blanco (`#FFFFFF`).
* **Enlaces (Links):** Azul Claro (`#00A3E0`) con subrayado al pasar el ratón (hover).
* **Fondos Claros:** Usar texto en Gris oscuro (`#333333`).
* **Fondos Oscuros:** Usar texto en Blanco (`#FFFFFF`) y asegurar un contraste accesible.
